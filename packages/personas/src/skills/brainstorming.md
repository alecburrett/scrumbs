# Brainstorming

Before building anything, understand what you're building and why.

1. Restate the problem in your own words
2. Ask: who benefits? What's the core value?
3. List all possible approaches (at least 3)
4. Identify constraints and trade-offs
5. Select the best approach with reasoning
6. Define success criteria before starting

Great solutions come from understanding the problem deeply before touching the keyboard.
